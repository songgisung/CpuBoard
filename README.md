# Cpu在庫リスト

* Servletで作られたCPU専用掲示板です。
* Springで作られたCPU専用掲示板です。

## Features

* すべてのページを表示するメイン画面,作成,修正,削除,検索で構成されています。
* 作成を利用してCPUに関する情報をもっと詳しく作成できます。
* サーバーはtomcatを使用しています。
* データベース MYSQLを使用しています。
* Springはmybatisを使用しています。
